# COVID-19 in Newfoundland and Labrador

This notebook studies a public data set on COVID-19 that was published online
by Health Canada. The data set records a regular count of the total number of
cases and deaths attributed to the disease in each of the Canadian provinces. The
objective is to use this data set in order to study the impact of the disease
upon Newfoundland and Labrador.
